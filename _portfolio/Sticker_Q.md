# Prompt
Produce a chibi-style LINE-inspired half-body character sticker set. Preserve the character’s correct headwear. Use a colored hand-drawn aesthetic. Create a 4×6 layout containing a wide range of common chat phrases and relevant entertainment memes. Do not replicate the source image. All text must be handwritten American English. Final output images must be 4K resolution at 16:9.

<img src='/images/Ty_Q_CN.jpeg'>

为我生成图中角色的绘制 Q 版的，LINE 风格的半身像表情包，注意头饰要正确。彩色手绘风格，使用 4x6 布局，涵盖各种各样的常用聊天语句，或是一些有关的娱乐 meme。其他需求：不要原图复制。所有标注为手写简体中文。生成的图片需为 4K 分辨率 16:9。
